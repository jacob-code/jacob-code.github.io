rspec-tree-runner

Analyze and run RSpec files with Github's Atom Editor

Atom package designed to improve the process of testing ruby files with RSpec.

The package will help you in this three ways:

  • Easily toggle between the tested file and the spec.
  • Display hierarchicaly the spec file, navigating easily through it.
  • Run the tests without leaving the editor, having the results integrated in the tree.

Install with apm install rspec-tree-runner.
